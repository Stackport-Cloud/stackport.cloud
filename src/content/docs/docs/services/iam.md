The IAM browser provides a read-only view of your identity and access management resources.

## Features

- **Users** — list with attached policies and group memberships
- **Roles** — trust policies and permission boundaries
- **Groups** — member users and attached policies
- **Policies** — JSON policy document viewer
- **Trust relationships** — visualize who can assume each role
- **Tagging** support

## Read-Only

The IAM browser is read-only. No create, update, or delete operations are available through StackPort.
