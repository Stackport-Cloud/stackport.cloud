The EC2 browser covers instances, networking, and storage resources.

## Features

- **Instance list** with state badges (running, stopped, pending, terminated)
- **VPC browser** — view VPCs and associated subnets
- **Subnet details** — CIDR, availability zone, available IPs
- **Security groups** — inbound/outbound rules
- **Volume information** — attached EBS volumes
- **Tagging** support
- **Export** resources as JSON or CSV

## Write Operations

| Operation | Description |
|---|---|
| Start | Start stopped instances |
| Stop | Stop running instances |
| Reboot | Reboot running instances |
| Terminate | Terminate instances permanently |
