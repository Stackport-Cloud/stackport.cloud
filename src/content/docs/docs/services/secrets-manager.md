---
title: Secrets Manager Browser
description: Browse secrets with value reveal, rotation status, and tagging.
---

The Secrets Manager browser lets you inspect secrets stored in AWS Secrets Manager.

## Features

- **Secret list** with pagination
- **Secret detail view** with reveal/hide toggle
- **JSON formatting** for structured secret values
- **Copy to clipboard** — copy secret values
- **Rotation status indicator** — see if rotation is enabled
- **Tagging** support

## Read-Only

The Secrets Manager browser is read-only. Secret creation and updates are done through your application or AWS CLI.
