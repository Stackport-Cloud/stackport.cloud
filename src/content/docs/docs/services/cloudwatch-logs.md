The CloudWatch Logs browser lets you explore log data with time-based filtering.

## Features

- **Log group list** with retention period and stored bytes
- **Log stream browser** — navigate streams within a group
- **Event viewer** with JSON parsing and syntax highlighting
- **Timestamp filters** — filter events by time range
- **Relative time display** — "5m ago", "2h ago"
- **Copy to clipboard** — copy individual log events
- **Pagination** for large log streams

## Read-Only

The Logs browser is read-only. Log ingestion happens through your application or emulator, not through StackPort.
